
# Changelog
All notable changes to this project will be documented in this file.

### [1.2.1] - 2018-03-20
## Added
- Addon versioning
- Support for limited edition addons (banner packs for events and such)
- Version coloring for easier understanding of what needs to be updated
## Changed
- Underlying structure of the updater to make it more flexible


### [1.1.1] - 2018-02-27
## Added
- Console window title TrollGame Updater version
- Colored console text
- Fixed some display misplacement
## Changed
- Adjusted logo position

### [1.1.0] - 2018-02-14
## Added
- New menus
- Feature to download troll-game addons
- Minor stability improvement
## Changed
- Backup archive name format now includes time stamp
- Reverted to py2exe packer due to anti-virus false positives when packing with pyinstaller

### [1.0.2] - 2018-02-12
## Added
- Executable icon
- Stability improvements
- Code comments
## Changed
- Smaller executable size thanks to pyinstaller


### [1.0.1] - 2018-02-08
## Changed
- Trollpatch download link


### [1.0.0] - 2017-10-25
## Added
- Download file feature
- Backup directory feature
- Unzip archive feature
- Remove file / directory feature
- Executable
