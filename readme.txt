
TrollPatch v1.2.1

# Simple patching & updating utility for Mount & Blade Warband NeoGK mod with TrollGame flavour.
# Written in Python 2.7 32bit and compiled to executable with py2exe 0.6.9 (python -m ensurepip | pip install py2exe_py2)

Instructions:

Place the troll_updater.exe file anywhere on the hard disk OUTSIDE of the Native directory.
Run it and follow the instructions on the screen. It should be pretty straight forward.
